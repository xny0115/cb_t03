모델 설정 및 관리

챗봇 모델의 학습 파라미터와 동작을 제어합니다.

기본 학습 설정

- 사용할 모델 구조를 선택합니다. (성능과 속도에 영향)
- 전체 학습 반복 횟수입니다. (높을수록 학습이 깊어짐)
- 한 번에 학습에 사용하는 데이터 수입니다. (메모리 부족 시 줄이세요)
- 모델이 얼마나 빠르게 학습할지를 결정합니다.
- 학습률을 천천히 증가시키는 초기 단계 수입니다.
- 한 문장에 허용되는 최대 토큰 수입니다.

모델 아키텍처

- 각 토큰이 표현되는 벡터 크기입니다.
- 내부 계산에서 사용되는 은닉층 크기입니다.
- 인코더 층의 개수입니다. (모델 깊이)
- 디코더 층의 개수입니다.
- 병렬 attention 처리 수입니다. (성능과 계산량에 영향)
0.10
- 과적합 방지를 위한 무작위 비활성화 비율입니다.
- 사용할 어텐션 메커니즘 종류를 선택합니다.
- 시퀀스 내 토큰 위치 정보를 인코딩하는 방법입니다.

환경 설정

- 입력 데이터에 적용할 전처리 기법을 선택합니다.
- 단어/토큰 임베딩의 벡터 크기입니다.
- 신경망 레이어에서 사용할 활성화 함수입니다.
- 모델 학습에 사용할 최적화 알고리즘입니다.
- 학습률 조정 전략을 선택합니다.
- 모델 학습의 안정성을 높이는 정규화 기법입니다.

추론 설정

- 확률 상위 K개 중에서만 단어를 선택합니다.
- 값이 클수록 더 다양하고 창의적인 답변이 생성됩니다.
- 누적 확률 P 이하의 후보들만 고려합니다.
- 답변 생성 시 고려할 후보 문장 수입니다.
- 답변의 다양성을 높이기 위한 페널티 값입니다.
- 반복적인 답변을 줄이기 위한 페널티입니다.
- 생성될 답변의 최대 토큰 길이입니다.

고급 설정

- 모델 성능 개선이 없을 시 학습을 조기 종료합니다.
- Early Stopping을 위한 성능 개선 대기 에폭 수입니다.
- 특정 대화 패턴을 인식하여 최적화합니다.
- 그래디언트 폭주를 방지하여 학습을 안정화합니다.
- 과적합을 줄이기 위한 정규화 기법입니다.
- 모델 체크포인트 저장 주기 (0은 에폭마다 저장).
- 데이터 로딩에 사용할 프로세스/스레드 수입니다.
- 데이터 로딩 시 GPU 메모리 고정을 활성화합니다.
- 학습 속도 향상 및 메모리 절약을 위해 혼합 정밀도를 사용합니다.
